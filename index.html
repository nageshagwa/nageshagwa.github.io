文本转换

计数是怎么算出来的

工具先按换行把文本切成行与段落，再逐个字符扫描：落在中日韩统一表意文字、假名、谚文等区段的字符按「中文字符」单独计 1；其余按空白切分出的连续片段算 1 个西文单词。这与 WPS、Microsoft Word「字数统计」里「中文字符和朝鲜语单词」+「英文单词」的口径一致，因此中英混排也能数准。

什么会被算进去

字符数（计空格）数所有字符，包括空格与标点；字符数（不计空格）去掉所有空白；中文字符只数表意文字与假名/谚文；段落按空行分隔。换行符 CR/LF、制表符等不可见字符也会影响字节数，但默认不计入「字数」。

常见问题

这个字数统计工具会上传我的文本吗？
不会。所有统计与转换都在你浏览器里用 JavaScript 本地完成，文本不会发送到任何服务器，关闭页面即清除。
中文字数是怎么算的？
落在中日韩统一表意文字、扩展区、假名与谚文区段的每个字符各计为 1 个「中文字符」，与 WPS / Word 的「中文字符和朝鲜语单词」口径一致。
「字数」和「字符数」有什么区别？
字符数数的是每一个字符（含空格、标点）；字数 = 中文字符数 + 西文单词数，更接近论文或稿费里说的「字数」。
和 WPS、Word 的字数统计结果一样吗？
对纯中文或纯英文文本通常一致；中英混排、含大量符号或全角字符时可能有个位数差异，因为各家对标点与空白的归类略有不同。
能统计英文单词数吗？
可以。去掉中文字符后，按空白切分出的连续片段即为西文单词，适合英文论文与摘要的词数限制核对。
UTF-8 字节数有什么用？
数据库字段、表单与接口常按字节限长。一个英文字母占 1 字节，一个中文字符通常占 3 字节，字节数能帮你避免超限被截断。

什么时候用得上

  • 核对论文 / 报告是否满足规定字数或词数限制。
  • 压缩简历到一页：盯住字符数与行数，删到合适篇幅。
  • 写社媒文案时控制单条字符上限，避免被截断。
  • 为 SEO 标题与描述把字符数控制在显示范围内。
  • 做大小写 / 首字母转换，快速规范英文标题与表头。
